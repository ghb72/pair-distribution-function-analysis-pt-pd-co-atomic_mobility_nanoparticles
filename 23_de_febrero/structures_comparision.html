import pandas as pd
import matplotlib.pyplot as plt

BCC = pd.read_csv('C:/Users/guill/OneDrive/Documentos/Tesis/Proyect/23_de_febrero/BCC/BCC.txt', sep= '\s+')
FCC = pd.read_csv('C:/Users/guill/OneDrive/Documentos/Tesis/Proyect/23_de_febrero/FCC/FCC.txt', sep= '\s+')
HCP = pd.read_csv('C:/Users/guill/OneDrive/Documentos/Tesis/Proyect/23_de_febrero/HCP/HCP.txt', sep= '\s+')
ICO = pd.read_csv('C:/Users/guill/OneDrive/Documentos/Tesis/Proyect/23_de_febrero/ICO/ICO.txt', sep= '\s+')

fig, ax =plt.subplots(figsize=(10,8), layout='constrained')
ax.plot(BCC['#r(A)'], BCC['G(r)'], color = 'black', label = 'BCC')
ax.plot(FCC['#r(A)'], FCC['G(r)'] +0.4, color = 'red', label = 'FCC')
ax.plot(HCP['#r(A)'], HCP['G(r)']+0.8, color = 'blue', label = 'HCP')
ax.plot(ICO['#r(A)'], ICO['G(r)']+1.2, color = 'green', label = 'ICO')


ax.set_xlabel('r [$\\AA$]')
ax.set_xlim([0,30])
ax.set_ylabel('G(r)')
ax.set_title('Comparación de PDFs para nanopartículas de 3nm con diferentes estructuras')
#ax.grid('True', linestyle='-')
ax.legend()
plt.show()

Podemos notar como la estructura HCP al tener el mayor porcentaje de llenado presenta menos picos pronunciados en comparación con las otras estructuras (exceptuando la estructura icosaédrica), se puede apreciar que los picos que aparecen en su gran mayoría presentan anchos mayores que los picos presentes en otras estructuras y están presentados en gran cantidad, coorespondiendose con el hecho de que existen mayor cantidad de vecinos interatómicos que la teoría explica en comparación con otras estructuras.
